# 🍅POMODORO SELF STUDY LAMP
A smart productivity tool that uses visual light cues to implement the Pomodoro Technique. This project transforms a standard lamp into a focus assistant, helping you manage work intervals and break times. It can be control by using our website with 5 minutes rest and also 25 minutes focus session.

## Objectives:
1. Enhance focus and productivity
2. Provide structured digital learning resources
3. Create an affordable smart studying tool
